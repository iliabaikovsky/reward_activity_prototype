# Чеклист: скопировал шаблон → новый проект

Используй этот список **в папке-копии** (не обязательно в workshop).

- [ ] Папка переименована под задачу (например `onboarding-prototype`).
- [ ] Открыта в Cursor как **корень проекта** (чтобы подхватились `.cursor/rules`).
- [ ] Заполнен **`DESIGN.md`**: ссылка на Figma, экраны, вьюпорт.
- [ ] В **`package.json`** изменено поле `name`.
- [ ] В **`index.html`** обновлён `<title>`.
- [ ] В терминале **в корне копии**:
  - [ ] `npm install`
  - [ ] `npm run dev` — проверка локально
  - [ ] `npm run build` — перед деплоем
- [ ] Деплой по **`DEPLOY.md`** (`npx vercel login` при первом разе).
- [ ] В чате Cursor: ссылка на Figma + что сверстать (можно отослаться к `DESIGN.md`).

**В самом репозитории workshop** шаблон можно держать без `node_modules` — зависимости ставишь только после копирования.
